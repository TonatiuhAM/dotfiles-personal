# Dotfiles

Configuraciones personales para mi entorno de desarrollo en macOS.

## Herramientas incluidas

- **zsh** - Shell configuration y aliases
- **tmux** - Terminal multiplexer setup
- **nvim** - Neovim configuration con LazyVim
- **ghostty** - Terminal emulator settings
- **aerospace** - Window manager configuration
- **karabiner** - Keyboard customization
- **fastfetch** - System information display
- **bash** - Bash configuration
- **opencode** - OpenCode editor settings

## Instalación

```bash
git clone https://github.com/tuusuario/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
```

Luego crear los symlinks necesarios para cada herramienta según sea necesario.

## Estructura

Cada directorio contiene la configuración específica para su herramienta correspondiente, organizada para mantener el directorio home limpio.
